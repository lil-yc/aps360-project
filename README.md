# AI-Generated Music Detection
## Overview
This project implements a CNN to detect whether a music clip was AI-generated or human-composed. 

## Files
- `models`: contains the saved model files
- `baseline.ipynb`: full pipeline for the baseline model
- `primary.ipynb`: full pipeline for the primary model

See `APS360_Project_Proposal.pdf`, `APS360_Progress_Report.pdf` for project details.

## Data
Data sourced from [this dataset](https://huggingface.co/datasets/SleepyJesse/ai_music_tiny).
